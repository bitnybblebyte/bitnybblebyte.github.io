The purpose of Bit.Nybble.Byte (BNB) is to provide an environment for students to expand their knowledge in Java based software design past the material covered by the programming courses offered in the school curriculum.

Java

BNB aims to expand knowledge in designing and creating software, focusing primarily on Java, beyond what is taught in school (e.g. game design, networking, OpenGL, etc.).

Git

Members can cooperate on projects by using Git, a version control system for collaborative coding.
